Projects

Control Logs Access Project

The team I worked for at IDVA was responsible for all of the facilities that offered training that could use Veteran benefits such as the Post 9/11 GI Bill. The types of training were varied: it could be anything from an On the Job Apprenticeship to a Non-Certificate Degree. Needless to say, the number of facilities, and veterans, was significant. As part of my time at IDVA, I was tasked with overhauling our team's data storage and reporting. Before this project, they were using multiple spreadsheets as a database and manually parsing data every quarter for their reporting. It was... not good, to say the least.
The Access database I created managed to achieve three things:
1. Simplified reporting down to the click of a button.
2. Provided a streamlined front-end for the users.
3. Integrated various reports into our own reporting.
This database has been in use for about a year now, and it's only managed to improve. I've had a few iterations to tweak the database as needed and added features that were requested, such as additional, more complex, reporting. This database has become essential to the team.

Pathfinder Combat Simulator

In my off time, I decided to work on a personal project of mine. In Pathfinder Second Edition, there are various classes to choose from. You can pick any class from a Fighter to a Rogue and so on. At some point, I asked myself:
"What class does the most damage?"
Although this was more of a fleeting thought than anything else, I found myself building a Java project in order to answer that question. I built this project with the three action system of Pathfinder in mind: each character has three actions per turn to use. Although you can use a single action to Strike, classes often provide varied ways to use your actions that can improve your damage. Due to the sheer amount of options each class has, I haven't come up with a concrete, singular answer yet. This continues to be a work in progress. However, this project has led to better balancing of certain classes that struggle due to system mechanics, such as the Swashbuckler and their Panache mechanic.
